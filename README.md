# 목탁: 번뇌파쇄록 — MOKTAK: Echoes of Nirvana

> "당신의 목탁은 이제 그냥 나무가 아닙니다."

목탁의 공명으로 번뇌를 때려부수는 **모바일 웹 3D 액션 어드벤처**.
백엔드 없음, 서버 없음, 로그인 없음 — 모든 것은 브라우저와 `localStorage` 안에서 울린다.

플레이어는 **공명의 수행자**가 되어 가상 조이스틱으로 이동하고,
목탁 타격·차지 공명·회피·스킬 3종으로 잡념 덩어리, 미루기의 망령, 알림중독 귀신을 정화하며
**게으름의 안개왕**부터 **마감임박 수라장군**까지 5종의 번뇌 보스를 상대한다.

## 실행 방법

```bash
npm install
npm run dev        # 개발 서버 (--host 로 같은 네트워크의 폰에서 접속 가능)
```

터미널에 표시되는 `Network:` 주소를 **모바일 브라우저**로 열면 된다.
(햅틱 진동은 Android Chrome 등 `navigator.vibrate` 지원 브라우저에서 동작)

프로덕션 빌드:

```bash
npm run build      # dist/ 생성
npm run preview    # 빌드 결과 로컬 서빙
```

정적 파일만 있으면 되므로 GitHub Pages / Netlify / 아무 정적 호스팅에나 올릴 수 있다.

## 조작

| 입력 | 동작 |
| --- | --- |
| 왼쪽 하단 드래그 | 가상 조이스틱 이동 (드래그 거리 비례 속도) |
| 木 버튼 탭 | 목탁 타격 — 금빛 공명파 + 넉백 |
| 木 버튼 길게 | **깊은 공명** 차지 → 손을 떼면 광역 충격파 |
| 회피 버튼 | 구르기 (짧은 무적, 위협 근처에서 성공 시 공명 게이지 보너스) |
| 스킬 버튼 3종 | 금강연타 · 만다라 폭진 · 해탈의 종 |

공명 게이지가 가득 차면 **해탈 모드** — 공격력/속도/범위 상승, 화면이 금빛과 청색으로 물들고 바닥에 거대한 만다라가 펼쳐진다.

## 구성

- **스테이지 6종**: 새벽 법당 → 비 오는 산사 → 달빛 연못 → 번뇌의 회랑 → 구름 위 종루 → 우주 만다라
- **보스 5종**: 게으름의 안개왕, 알림중독의 악귀, 비교지옥의 거울, 카드값의 아수라, 마감임박 수라장군
- **성장**: 수행력/공명석/업보로 9종 강화 (손목 스냅 수련, 법당 리버브 증폭, 방석 중심 안정화…)
- **유물 10종**: 낡은 염주, 금빛 방석, 달빛 목탁, 연꽃 부적… (보스 정화 보상, 패시브 효과)
- **사운드**: 외부 파일 0개 — 목탁음/종소리/정화음 전부 Web Audio API 합성
- **저장**: `localStorage` (진행도, 재화, 강화, 유물, 설정, 기록. 설정에서 초기화 가능)

## 기술

React + TypeScript + Vite + Three.js(@react-three/fiber, drei) + Zustand + Web Audio API.

- 게임 시뮬레이션은 React 밖의 가변 월드 상태(`src/game/world.ts`)에서 `useFrame` 루프로 돌고,
  HUD는 8Hz로만 동기화해 리렌더를 최소화한다.
- 외부 3D 모델/이미지 에셋 없음 — 모든 비주얼은 기본 지오메트리, 절차적 캔버스 텍스처(만다라),
  가산 블렌딩 파티클, 조명으로 구성.
- 설정에서 그래픽 품질(낮음/보통/높음) 선택: DPR, 파티클 수, 그림자가 조절된다.

```
src/
  game/    타입·밸런스·월드 시뮬레이션·전투·적/보스 AI·오디오·햅틱·저장·스토어
  three/   R3F 씬 (플레이어, 적, 보스, 스테이지, 파티클, 충격파, 만다라, 카메라)
  ui/      HUD, 가상 조이스틱, 액션/스킬 버튼, 허브(강화/유물/스테이지/설정), 오프닝
```
